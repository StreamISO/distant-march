你可以键入代码, 通过编程的方式操作游戏
下面是所有可用指令 [XXX]表可选:

reset threshold [nowait]; //在获得马赫点数达到 threshold 时重置, 若有 nowait, 则不满足条件时跳过此语句, 否则等待 (后面 nowait 意义相同)
E.g: reset 1.00e30; //等待单次重置能获得 1.00e30 马赫点数时重置

star threshold [nowait] [respec]; //在星辰点数达到 threshold 时流星雨, 若有 respec, 则重置星辰链
E.g: star 32 respec; //等待能获得 32 星辰点数时流星雨, 并重置星辰链

purchase [nowait] x1, x2, ..., xn; //购买星辰链 x1~xn 项升级
E.g: purchase nowait 01, 11, 13, 23; //购买星辰链 Start, 11, 13, 23. 若不成功, 直接跳过

restriction c1, c2, c3, c4; // c1~c4 = 0,off / 1,on. 将第 1~4 个限制选择状态设为 c1~c4
E.g: restriction on on on on; //勾选所有限制

challenge which [nowait] [star]; //进行第 which 个挑战, 若有 star, 则为星辰挑战
E.g: challenge 4 star; //进行星辰挑战4

matrix threshold; //在维度等级达到 threshold 时矩阵覆写
E.g: matrix 0; //能矩阵覆写时立刻执行

auto type [c] [value]; //c= on / off, 默认 on.将 type 自动购买设为 c , 若有数值, 则将此自动购买的数值改为 value
E.g: auto timer 5.00; //设置定时重置 5s/次
所有合法的 type: booster->推进器, counter->计数器, turbo->涡轮机, flow->整流罩, reset->自动重置, stellar->恒星引擎, mach42->自动购买音速升级42, timer->定时重置

log text; //在浏览器控制台输出 text

按键说明: Run->运行代码, 再点一次暂停 Stop->停止运行, 回到第一行 Loop->循环执行代码 MAST->每次矩阵周目开始时自动运行
